# Как запустить

### Backend:
``` docker-compose up -d --build
docker exec -it teammanager bash
composer require laravel/reverb
composer install
php artisan reverb:install
php artisan migrate
docker start teammanager_reverb
```
### Frontend:

```
cd frontend
npm i
npm run dev
```
